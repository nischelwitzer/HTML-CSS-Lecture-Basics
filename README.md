### Template HTML5 Page
```
<!DOCTYPE HTML>
<html lang="de">
	<head>
		<meta http-equiv="Content-Type" content="text/html; charset=utf-8"> 
		<title>My first HTML5 Title</title>
	</head>

	<body>
		My <b>first correct</b> HTML5 Page
		<p>HTML5 Hello World</p>
		<ul>
			<li>Text HTML5 with List</li>
			<li>Publishing: Server/FTP</li>
		</ul>
	</body>
</html>
```
